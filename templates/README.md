# Templates

`master_bom_attributes.xlsx` is auto-generated by `utils/template_generator.py` the first
time any journal runs. Do not commit a binary xlsx to version control — the file is already
excluded by `.gitignore` and should be generated at runtime.

## Regenerating manually

```
"<NX_ROOT>\NXBIN\python.exe" utils/template_generator.py
```

Running the script directly creates `templates/master_bom_attributes.xlsx` with the
`MASTER_COLUMNS` header pre-styled (bold, white text, `#1a1a2e` background).
